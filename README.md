# comfyui-manager-
comfyui manager汉化中文

直接将js文件放到你comfyui的这个路径\ComfyUI\custom_nodes\ComfyUI-Manager\js下，然后刷新一下界面就行ctrl+F5。

这个是基于https://github.com/fengmexp-cell/ComfyUI-Manager-Chinese-Translator-ZH-TW 改的，他这个文字改的不利于简体中文理解，所以我加工了一下。
